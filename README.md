# Smart Biometric Attendance System using ESP32

## Overview

This project is an IoT-based Smart Biometric Attendance System developed using the ESP32 microcontroller and an optical fingerprint sensor. It securely verifies user identities through fingerprint authentication, records attendance with timestamps, displays attendance records on a built-in web server, and automatically sends attendance notifications via email.

The project is designed for schools, colleges, offices, and organizations requiring an accurate and automated attendance management system.

## Key Features

- Fingerprint-based biometric authentication
- User enrollment and fingerprint matching
- Real-time attendance recording
- Web Server Dashboard for attendance monitoring
- Automatic Email Notification after attendance
- Date and Time logging
- Wi-Fi connectivity
- Fast and secure verification
- Easy to expand with cloud services like Firebase or Google Sheets
